---
title: Simple vs Advanced mode
description: A viewing preference for your sidebar and detail pages.
---

Guhan has two view modes. Both show the same underlying data — the difference is how much of the plumbing is visible in the sidebar.

## Simple mode (default)

Meant for founders + small teams doing outbound solo. Sidebar shows:

- **Home** — dashboard
- **Inbox** — task queue
- **Meetings** — upcoming calls
- **Outreach Agents** — your messaging campaigns
- **Companies** + **Prospects** — records
- **Brand & Voice** — how Guhan sounds
- **Workspace settings**

Behind the scenes, Lists and Watchlist Agents still exist and still work — they're just tucked into the Outreach Agent detail page as sub-tabs rather than nav-level items.

## Advanced mode

Reveals two extra sidebar rows:

- **Watchlist Agents** — see + edit them as standalone records (not just via the Outreach Agent detail page)
- **Lists** — see + edit them as standalone records too

Also reveals:
- Bulk-editing shortcuts on the Prospects page
- More detailed sequence canvas controls

Best for teams running many campaigns or intricate list-management workflows.

## Switching modes

Bottom of the sidebar: click the **Simple / Advanced** pill. Takes effect immediately (no reload). Your preference is per-user — teammates see their own.

## Auto-upgrade

Guhan will nudge you from Simple → Advanced automatically when it detects:

- **Multi-list audiences** — your Outreach Agent pulls from more than one List
- **Exclusion lists** — you're excluding prospects from another List
- **Shared graphs** — the same sequence version powers multiple Outreach Agents

When any of these lands, you'll see a one-time banner explaining the switch. Dismiss it and stay on Advanced.

## Simple mode limitations

Some things you can't do in Simple:

- Add multiple Lists to an Outreach Agent's audience (Advanced only)
- Configure exclusion lists (Advanced only)
- Share the same sequence version across multiple Outreach Agents (Advanced only)

If you try any of these, Guhan will offer to flip you to Advanced.

## Which mode should you use?

- **Solo founder / 1-person team** → Simple. Less to look at, faster to navigate.
- **Multi-persona outreach** (3+ Watchlist Agents feeding different sequences) → Advanced.
- **Team of 3+ with dedicated list managers** → Advanced.

You can switch back and forth anytime — no data lost either way.
