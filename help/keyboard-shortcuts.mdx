**All Guhan shortcuts live inside the sequence canvas — the visual builder for message flows. The rest of the app is mouse-driven for now.**

Press `?` (or `Cmd + /` on Mac, `Ctrl + /` on Windows) anywhere on the canvas to pull up this list in-app.

## Sequence canvas

Open any Outreach Agent → **Sequence** tab → the canvas. Press `?` or `Cmd + /` (macOS) / `Ctrl + /` (Windows) anytime to bring up this same list in-app.

<Note>
  The modifier key adapts to your OS. `Cmd` on macOS, `Ctrl` on Windows and Linux. The examples below use `Cmd` — swap for `Ctrl` if you're not on Mac.
</Note>

| Shortcut | Action |
|---|---|
| **Drag from palette** | Add a step to the canvas |
| **Click a step** | Open its configuration sheet |
| **Backspace** or **Delete** | Remove the selected step or arrow |
| **Shift + click** | Select multiple steps |
| **Cmd + D** | Duplicate selection |
| **Cmd + Z** | Undo |
| **Cmd + Shift + Z** | Redo |
| **Cmd + C** | Copy selected steps (works across canvases in the same browser) |
| **Cmd + V** | Paste steps |
| **Mouse wheel** OR **+/- buttons** (lower-right) | Zoom canvas |
| **Cmd + K** | Search steps by label or type |
| **?** or **Cmd + /** | Show this help modal in-app |

## Anywhere else in the app

No dedicated keyboard shortcuts yet. Standard browser shortcuts work everywhere (`Cmd + F` finds on page, `Cmd + Enter` submits forms where they exist, etc.).

If you'd like a specific shortcut added (e.g., `G` then `I` to jump to Inbox), email hello@guhan.ai — we prioritize by user request.
